# Python Memory Game

A terminal-based memory matching game with 3 difficulty levels:

- **Easy**: 2x2 board (2 pairs)
- **Medium**: 4x4 board (8 pairs)
- **Hard**: 4x6 board (12 pairs)

## Run

```bash
python memory_game.py
```

## How to play

1. Choose a difficulty level.
2. Pick two card positions each turn using `row col` format (example: `2 3`).
3. Match all pairs to win.
4. Enter `q` at a card prompt to quit.
